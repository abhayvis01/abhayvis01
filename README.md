## 🚀 Featured Projects
### 💳 LEDGER
*A production-ready financial management API designed to streamline expense tracking and account management.*
* **Stack:** Node.js, Express.js, MongoDB, JSON Web Tokens (JWT)
* **Key Accomplishments:**
  * Implemented secure token-based authentication and session state management.
  * Designed complex relational schemas within MongoDB to optimize querying performance for high-frequency transactions.
  * Constructed RESTful endpoints for ledger reconciliation, transaction logs, and analytical reporting.
### 🔑 OAuth Authentication API
*An enterprise-grade, standard-compliant authorization service focusing on security and token rotation.*
* **Stack:** Node.js, Express.js, JWT, OTP Verification Services
* **Key Accomplishments:**
  * Engineered a dual-token architecture utilizing short-lived access tokens and secure, rotation-enabled refresh tokens.
  * Integrated multi-factor authentication via secure OTP verification flows.
  * Developed robust security middlewares preventing CSRF, replay attacks, and token interception.
### 👥 Virtual Study Group
*A collaborative learning platform facilitating group interactions, resource sharing, and study room orchestration.*
* **Stack:** Flask, Python, MySQL, HTML5, CSS3
* **Key Accomplishments:**
  * Created dynamic virtual rooms utilizing a relational MySQL schema optimized for user joins and session scheduling.
  * Built a user dashboard displaying peer activity feed and material sharing capabilities.
  * Developed modular, clean MVC components within Flask to ensure scalability of features.
### 🎓 Digital Learning Platform
*An online educational dashboard designed to manage course progress, syllabus tracking, and evaluations.*
* **Stack:** Flask, Python, SQLite/MySQL, Jinja2 Templates
* **Key Accomplishments:**
  * Implemented role-based access control (RBAC) separating student and instructor capabilities.
  * Built progress trackers displaying course completion percentages dynamically using backend mathematical computations.
  * Deployed a responsive frontend styling coupled with scalable backend control loops.
